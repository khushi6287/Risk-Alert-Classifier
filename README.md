# 🛡️ Risk Alert Classifier using Machine Learning

## 📌 Project Overview

This project aims to develop a machine learning-based Risk Alert Classifier that predicts whether a customer is **High Risk** or **Low Risk**. The project compares multiple classification models and applies data preprocessing, imbalanced data handling, and hyperparameter tuning to improve prediction performance.

---

## 🎯 Objectives

- Predict customer risk levels using machine learning.
- Handle missing values and class imbalance.
- Compare different classification algorithms.
- Evaluate models using classification metrics.
- Select the best-performing model.

---
